**Spatial Join and Distance Calculation Script**
=====================================================

### Description

This script loads two shapefiles, performs a spatial join, and calculates the exact distance from every property to the nearest golf course boundary.

### Required Packages

* `sf` for simple features and spatial operations
* `dplyr` for data manipulation

### Script

```r
# Load required libraries
library(sf)
library(dplyr)

# Load shapefiles
load_properties <- function() {
  properties <- read_sf("properties.shp")
  return(properties)
}

load_golf_courses <- function() {
  golf_courses <- read_sf("golf_courses.shp")
  return(golf_courses)
}

# Perform spatial join and calculate distance
calculate_distance <- function(properties, golf_courses) {
  # Perform spatial join
  joined_data <- properties %>%
    st_intersection(golf_courses) %>%
    st_distance(by_me = properties, by_geom = golf_courses)
  
  # However, st_intersection returns only intersecting geometries.
  # For non-intersecting geometries, st_distance returns NA.
  # Instead, use st_nearest to find the nearest golf course for each property
  nearest_golf_courses <- properties %>%
    st_nearest(golf_courses, k = 1)
  
  # Calculate distance to nearest golf course
  dist_to_golf_m <- st_distance(properties, nearest_golf_courses$geometry) %>%
    as.numeric() %>%
    set_units(meters)
  
  # Add distance as new column
  properties$dist_to_golf_m <- dist_to_golf_m
  
  return(properties)
}

# Main function
main <- function() {
  properties <- load_properties()
  golf_courses <- load_golf_courses()
  updated_properties <- calculate_distance(properties, golf_courses)
  return(updated_properties)
}

# Run the main function
updated_properties <- main()
print(updated_properties)
```

### Usage

3. Run the script using R or RStudio.

The script will load the shapefiles, perform a spatial join, calculate the distance from every property to the nearest golf course boundary, and return the updated dataframe with the new column `dist_to_golf_m`.
